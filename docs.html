Getting Started

Welcome to the Markdown Editor! This powerful tool allows you to create, edit, and preview Markdown documents in real-time. Follow these simple steps to get started:

  1. Navigate to the Editor: Click the "Editor" link in the navigation menu or the "Start Editing" button on the home page.
  2. Start Writing: Begin typing your Markdown content in the left panel. The preview will automatically update on the right.
  3. Use the Toolbar: Access common formatting options from the toolbar above the editor.
  4. Save Your Work: Your content is automatically saved to browser storage. You can also export it as a .md file.
# Welcome to Markdown Editor This is a **paragraph** with some *emphasized* text. ## Features List - Live preview - Syntax highlighting - Auto-save - Export functionality

Basic Formatting

Markdown uses simple syntax to format text. Here are the most common formatting options:

Text Emphasis

**Bold text** or __Bold text__ *Italic text* or _Italic text_ ***Bold and italic*** or ___Bold and italic___

Headings

# Heading 1 ## Heading 2 ### Heading 3 #### Heading 4 ##### Heading 5 ###### Heading 6

Lists

- Item 1 - Item 2 - Nested item - Another nested item - Item 3 1. First item 2. Second item 3. Third item 1. Nested item 2. Another nested item

Links and Images

[Link text](https://example.com) ![Alt text](https://example.com/image.jpg)

Code

Use `console.log()` for debugging ```javascript function hello() { console.log("Hello, World!"); } ```

Blockquotes

> This is a blockquote. > It can span multiple lines. > > > Nested blockquotes are also supported.

Advanced Features

Our editor supports GitHub-flavored Markdown (GFM) with additional features:

Task Lists

- [x] Completed task - [ ] Incomplete task - [ ] Another incomplete task

Strikethrough

~~This text is struck through~~

Horizontal Rules

Keyboard Shortcuts

Work more efficiently with these keyboard shortcuts:

Ctrl + B - Bold text
Ctrl + I - Italic text
Ctrl + H - Insert heading
Ctrl + U - Unordered list
Ctrl + O - Ordered list
Ctrl + Q - Blockquote
Ctrl + K - Insert link
Ctrl + T - Insert table
Ctrl + S - Save document
Ctrl + ` - Inline code
Ctrl + Shift + C - Code block
Ctrl + Shift + I - Insert image
Tab - Indent list item
Shift + Tab - Unindent list item

Exporting Documents

Export your Markdown documents with these simple steps:

  1. Click the "Export" button in the header (download icon)
  2. The file will be automatically downloaded with a timestamp in the filename
  3. The exported file will have a .md extension and can be opened in any Markdown editor

Note: Your work is automatically saved to browser storage, so you can close and reopen the editor without losing your content.

Tips & Tricks

1. Use the Toolbar

The toolbar provides quick access to all formatting options. Hover over each button to see keyboard shortcuts.

2. Auto-Save

Your content is automatically saved every second. The "Last saved" timestamp shows when your work was last preserved.

3. Character Count

Track your document length with the character counter in the bottom-right corner of the editor.

4. Responsive Design

On mobile devices, the editor and preview panels stack vertically for better usability.

5. Copy-Paste

You can paste Markdown content from other sources directly into the editor. The preview will update automatically.

6. Syntax Highlighting

Code blocks automatically detect the programming language and apply appropriate syntax highlighting.
